# Stapler Reverse Engineering
📎 SolidWorks Stapler Project: Reverse Engineering & Design Improvement
This project showcases the complete reverse engineering and redesign of an Amazon Basics office stapler using SolidWorks. The goal was to disassemble a physical product, understand the function of each component, and recreate it digitally with precision and realism.

🔧 What I Did
Disassembled a real stapler and studied the purpose and interaction of each part

Measured all components accurately using calipers

Modeled every part in SolidWorks with full detail

Applied GD&T (Geometric Dimensioning and Tolerancing) to each part

Assembled the parts with dynamic constraints and three different types of mechanical fits

Improved the original design with modifications focused on ergonomics, weight reduction, and manufacturability

Created high-quality renderings and an exploded-view animation of the assembly

📁 What's Included
3D models of all original and redesigned parts

Working technical drawings with manufacturing details

Rendered images and animation files

Justification for design improvements

This project demonstrates my ability to work through the entire design process—from physical product analysis and measurement to 3D modeling, assembly, tolerancing, and design iteration. It reflects strong skills in mechanical design, reverse engineering, and SolidWorks.
